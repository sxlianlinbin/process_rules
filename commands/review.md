# Code Review Command

## 指令描述
此命令用于检查和审查修改的代码，帮助识别潜在问题、改进建议和最佳实践。

## 执行流程
1. 进入process项目
1. 获取当前修改的文件列表，并列出
2. 分析代码变更内容
3. 检查代码质量和规范
4. 提供改进建议

## 审查要点
- **代码规范**: 检查命名规范、代码风格、格式化
- **逻辑正确性**: 验证业务逻辑是否正确
- **性能优化**: 识别潜在的性能问题
- **安全性**: 检查安全漏洞和风险
- **可维护性**: 评估代码的可读性和可维护性
- **测试覆盖**: 确认是否需要添加或更新测试

## 输出格式
- 问题列表（按严重程度分类）
- 具体的改进建议
- 代码示例（如适用）
