完整数据库表结构文档

本文档展示了系统中所有数据库表的结构信息，包含字段详情和表之间的关联关系。

app_attribute - 事项属性:校区、办理形式等

字段名 类型 长度 可否空 注释 主外键
id serial NOT NULL 主键 PRIMARY KEY
name varchar 64 NOT NULL 名称
type smallint NOT NULL 类型
dict_id int NOT NULL 码表
max_len int NOT NULL 填写数据最大长度
required smallint NOT NULL 是否必填

app_category - 事项服务分类

字段名 类型 长度 可否空 注释 主外键
id serial NOT NULL 主键 PRIMARY KEY
name varchar 64 NOT NULL 分类名称
sort int NOT NULL 排序
modified timestamp(0) NOT NULL 修改时间
enable smallint NOT NULL 是否启用

app - 事项表

关联关系:
• depart_id → department.id (外键关联)
• creator → users.id (外键关联)
• public_version_id → app_version.id (外键关联)
字段名 类型 长度 可否空 注释 主外键
id serial NOT NULL ID PRIMARY KEY
name jsonb NOT NULL 名称
short_name jsonb NOT NULL 简称
icon varchar 128 NOT NULL 图标
type smallint NOT NULL 类型(1表单事项 2流程事项 3第三方事项)
public_version_id int NOT NULL 已发布版本id FOREIGN KEY → app_version.id
status smallint NOT NULL 状态(0草稿 1未发布 2待审核 3已发布 4隐藏)
keywords varchar[] NOT NULL 搜索关键词
depart_id int NOT NULL 所属部门 FOREIGN KEY → department.id
categorys int[] NOT NULL 服务分类
identity int[] NOT NULL 服务对象
visible_roles varchar[] NOT NULL 可见角色
apply_roles varchar[] NOT NULL 可申请角色
description text NOT NULL 申请说明
attributes jsonb NOT NULL 事项属性
star numeric(2,1) NOT NULL 事项评分
sort int NOT NULL 排序
agent_id int NOT NULL 发消息agent_id
creator int NOT NULL 创建人 FOREIGN KEY → users.id
created timestamp(0) NOT NULL 创建时间
modified timestamp(0) NOT NULL 修改时间
